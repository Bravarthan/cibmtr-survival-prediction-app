# CIBMTR Survival Prediction App

Machine Learning web application built using LightGBM and Streamlit to predict post-HCT survival outcomes.

## Tech Stack
- Python
- LightGBM
- Streamlit
- Pandas
- Scikit-learn

## Files
- app.py (Web Application)
- survival_prediction_model.pkl (Trained Model)
- requirements.txt
- CIBMTR notebook (.ipynb)

## Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
```
